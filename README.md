# SimpleCMakeList
This project shows how to create a very simple CMakeLists.txt file and compile a C/C++ project.
The directory structure of the Project is as follows:

```SimpleCMakeList
├── CMakeLists.txt
├── Examples
│   └── main.c
├── headers
│   ├── cube.h
│   └── square.h
├── LICENSE
├── README.md
├── sources1
│   └── cube.c
└── sources2
    └── square.c```


Usuage

##Requirements

1. The GNU toolchain: 
2. cmake
3. GNU make

##Instructions

###Building

1. Clone the repository: https://github.com/MrPoudel/SimpleCMakeList.git 

2. cd SimpleCMakeList

3. mkdir build

4. cd build && cmake ..

5. make

6. ./main
